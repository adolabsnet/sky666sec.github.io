kali安装GDebi程序包管理器

kali

dpkg是一个功能强大的工具，但它并不自动安装依赖项。为此，我们需要某种程序包安装工具，以便在安装.deb程序包的同时，可以去获取所有必要的依赖项。眼下最出色的程序包安装工具非gdebi莫属。

使用的命令来安装它：

apt-get update -y

apt-get install gdebi -y
